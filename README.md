# Gmail Attachment Automation 📩➡️📂

This project automates saving Gmail attachments directly into Google Drive using Google Apps Script.

## 🚀 Features
- Automatically detects unread emails with attachments
- Saves attachments to a specified Google Drive folder
- Marks emails as read after processing

## 🛠️ Technologies Used
- Google Apps Script
- Gmail API
- Google Drive API

## 📌 How It Works
1. The script scans Gmail for unread emails with attachments
2. Downloads the attachments
3. Stores them in a selected Google Drive folder
4. Marks emails as read

## ▶️ How to Use
1. Open Google Apps Script
2. Paste the code
3. Add your Google Drive folder ID
4. Run the script
5. Set a trigger for automation

- Integrated Gmail and Google Drive
